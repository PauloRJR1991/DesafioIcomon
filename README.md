# DesafioICOMON
Para fazer o projeto, eu utilizei MVC com Razor Pages porque essa combinação facilita separar a lógica da aplicação (Model e Controller) da interface visual (Views/Razor Pages). O MVC me deu organização e estrutura para lidar com regras de negócio e banco de dados, enquanto as Razor Pages tornaram mais rápido e prático criar telas com formulários e tabelas. Assim, consegui manter o projeto bem estruturado e, ao mesmo tempo, produtivo no desenvolvimento da interface.”

O que foi feito:
* Montei todos os MVCs completos, com regras.
* Exportação em excel dos funcionario.
* Organograma.

Para executar o projeto deverá ser aplicado os scripts em(na sequência):
 * DesafioICOMON\DesafioICOMON\Scripts\Tabelas\
 * DesafioICOMON\DesafioICOMON\Scripts\Procedures\
Adicionar a ConnectionStrings:
 " DefaultConnection": "Server=Informe_Servidor;Database=Informe_BancoDeDados;Security=True;TrustServerCertificate=true;MultipleActiveResultSets=true"
   Substituir "Informe_Servidor": Deve substituir pelo seu servidor;
   Substituir "Informe_BancoDeDados": Deve substituir pelo seu banco de dados;
